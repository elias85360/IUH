# IoT Dashboard (Full Stack)

Folders:
- `backend/` – Node.js (Express + Socket.IO) with mock ingestion
- `frontend/` – React + Vite dashboard UI
- `docs/` – API, Security, and Frontend docs

Quick Start:
1) Backend
```
cd backend
cp .env.example .env   # optional
npm install
npm start
``` 

2) Frontend (in new terminal)
```
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173`.

Docs:
- API: `docs/API.md`
- Security: `docs/SECURITY.md`
- Frontend: `docs/FRONTEND.md`
